# ARQ.O — Estudio de arquitectura

Sitio institucional/portfolio minimalista para un estudio de arquitectura, construido con Next.js (App Router), TypeScript y Tailwind CSS.

## Scripts

- `npm run dev` — entorno de desarrollo
- `npm run build` — build de producción
- `npm run start` — servidor de producción
- `npm run lint` — linting
